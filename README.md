# Virtualscroll

Es un proyecto generado con [Angular CLI](https://github.com/angular/angular-cli) version 9.1.0.

## Herramientas

Se usa el Drag and Drop del CDK [Drag and Drop](https://material.angular.io/cdk/drag-drop/overview)
